Answer: Javascript is an interpreted programming language.It interpreted line by line.Javascript works in browser with the help of javascript engine.Different browser have different engine.Javascript works in chrome browser with the help of javascript engine which called v8 engine.this engine is written in c++.v8 engine which execute code with the help of jit compiler.Chrom v8 engine parse javascript source code,then abstract syntax tree,it converts into Byte code,then machine code.Finally code runs to the CPU.This is how Javascript work.

Answer: Local storage and Session Storage works in a same way.Both storage has 4 methods.But there are some difference between local storage and session storage.Local Storage store data.When the browser tab or window is closed,the data which store in local storage does not expire.It remains,data is not lost.On the other hand,In session storage,when the browser tab is closed ,data will be lost.but we can set value in the same way in both storage.We can store single value easily.But if we want to store object or array in both storage ,it have to convert into string with JSON.stringify,then will be stored in storage.
